<H1>VaCalendar</H1>
<h2>VaCalendar - a simple bot for planning vacations.</h2>

<h3>Requirements:</h3>
 - aiogram
 - APScheduler
 - python-decouple
 - redis
 - asyncpg

<h3>Install:</h3>
- <code>gh repo clone wratixor/vacalendar</code>
- <code>pip install -r requirements.txt</code>
- Edit template.env and rename to .env
